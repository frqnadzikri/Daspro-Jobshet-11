2.  Apa fungsi dari Arrays.toString()?
- Arrays.toString() adalah method yang digunakan untuk mengubah array menjadi String dalam format yang mudah dibaca.
3. Apa nilai default untuk elemen pada array dengan tipe data int?
- Nilai default untuk array tipe int adalah 0 (nol).
Ketika kita membuat array int baru seperti new int[5], semua elemennya otomatis diisi dengan nilai 0.
5.  Array myNumbers memiliki length berbeda untuk setiap barisnya. Apakah panjang array dapat dimodifikasi setelah diinstansiasi?
- TIDAK, panjang array TIDAK bisa dimodifikasi setelah diinstansiasi.
Setelah kita membuat array dengan ukuran tertentu, ukurannya tetap/fixed dan tidak bisa diubah.
